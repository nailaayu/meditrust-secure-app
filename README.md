# MEDITRUST SECURE APP

## Bahasa Indonesia

Website rekam medis berbasis PHP dan MySQL dengan pengamanan terhadap kerentanan OWASP Top 10 serta enkripsi data menggunakan AES-256-CBC.

### Fitur
- Pengelolaan rekam medis pasien
- Pengamanan OWASP Top 10
- Enkripsi data AES-256-CBC

### Teknologi
PHP, MySQL, Bootstrap, dan CSS.

### Cara Menjalankan
Simpan proyek pada folder `htdocs` atau `www`, impor database, lalu jalankan Apache dan MySQL.

---

## English

A medical record website developed with PHP and MySQL, featuring protection against OWASP Top 10 vulnerabilities and AES-256-CBC data encryption.

### Features
- Patient medical record management
- OWASP Top 10 security protection
- AES-256-CBC data encryption

### Technologies
PHP, MySQL, Bootstrap, and CSS.

### How to Run
Place the project in the `htdocs` or `www` folder, import the database, then start Apache and MySQL.
